# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website developed using **HTML5, CSS3, and JavaScript**. This portfolio showcases my education, technical skills, certifications, projects, experience, and contact information with a clean and professional user interface.

## 🚀 Live Demo

🌍 Live Website:
https://nadeem7n.github.io/portfolio-website/

## 💻 GitHub Repository

https://github.com/nadeem7n/portfolio-website

---

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern blue-green gradient user interface
- Professional Home section
- About Me section
- Technical Skills with progress bars
- Education section
- Experience section
- Certificates section
- Project Showcase
- Download CV feature
- Social media links
- Working Contact Form using EmailJS
- Smooth scrolling navigation
- Typing animation
- Scroll reveal animations
- Scroll-to-top button

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- EmailJS
- Font Awesome
- Google Fonts
